# 伺服器身份組介紹

# 基本身份組

# @蘿莉控

  • 所有人都有的身份組，拿到了才可以進入避難所內聊天
  • 獲取方法：進入之後填寫自介，管管審核通過之後便可獲取

# @很嗨的蘿莉控

  • 啊就很嗨就能拿到了
  • 獲取方法：Atlas等級達到三級，可以使用a!rank指令來查詢等級

# @管管

  • 伺服器的最高級管理員，擁有至高無上的權利
  • 獲取方法：管理層內部選拔，而且需要來自所有管管的認可，及對本伺服器做出過卓越的貢獻

# @伺服助理

  • 比管管低的管理員，協助管管
  • 獲取方法：接受申請，不過依然是以管理層選拔為主
  • 申請條件：沒有嚴重觸犯過伺服條規，受到管理層的信任和認可，顯示出可靠的管理能力

# @圖片管理員

  • 管理圖片啦
  • 獲取方法：接受申請
  • 申請條件：至少要有 @圖庫數據庫

# @技術人員

  • 會幫忙處理機器人以及給予伺服器設置建議
  • 獲取方法：找已有身份組的人認證，最少要有伺服助理的身份

# @機器倫

  • 啊就機器人啊
  • 獲取方法：啊你要這個幹什麼鬼

# @dj

  • 聽歌用
  • 獲取方法：接受申請
  • 申請條件：啊你來申請就有了啦，不過需要注意不要亂跳別人點的歌，一經拔除永不撤回

# @合作

  • 與避難所合作的群主
  • 獲取方法：來合作啊
  • 合作條件：私訊管管/助理，管理層進入群組觀察之後再討論要不要合作

# @老司機

  • 色痞專用
  • 獲取方法：接受申請，或者管管覺得你很色也會給

# @地窖守護者

  • 在 #地窖 嗨的就有了啦
  • 獲取方法：接受申請

# @地窖的奴隸

  • 抖M專屬
  • 獲取方法：接受申請

# @老婆們の霸主

  • Mudae機器人區獲取至少10位老婆
  • 獲取方法：接受申請

# @色文大師

  • 發自己寫的色文就有啦
  • 獲取方法：接受申請

# @正太

  • 男孩子
  • 獲取方法：開放申請

# @蘿莉本人

  • 女孩子
  • 獲取方法：開放申請

# @蘿莉貓娘

  • 跟惡魔貓男有關係的人
  • 獲取方法：和惡魔貓男組成情侶即可跟管管申請

# @惡魔貓男

  • 跟貓娘有關係的人的身份組
  • 獲取方法：和貓娘組成情侶關係即可跟管管申請

# @有錢的乾爹

  • 加成服務器的好野人
  • 獲取方法：加成服務器自動獲得

# @蘿莉圖來源商

  • 在 #蘿莉圖 發送25張以上的蘿莉圖就有啦
  • 獲取方法：接受申請，或者管管給

# @vtuber圖片來源商

  • 在 #vtuber區 發送25張以上的圖片
  • 獲取方法：接受申請，或者管管給

# @圖庫批發商

# @圖庫數據庫

  • 獲取方法：開放申請由管管核定

# @圖庫之神

  • 獲取方法：自己申請由管管核定
