# 服務器條款

1.這是一個關於蘿莉的群組，來這裡的大家應該也都是為了蘿莉而來，請不要在這裡宣傳或者吵架

2.請不要不斷@tag騷擾人

3.不要錯頻發文

4.不要刷頻

5.禁止三次元蘿莉

6.不要獵奇，可能有些人會覺得不舒服

7.請遵從管理員們的指示

8.加小號請向管理員通報

9.請多用群組貼圖，如果覺得貼圖不足，可在 #表情建議區 提出建議

10.歡迎大家邀請更多的蘿莉控進來
